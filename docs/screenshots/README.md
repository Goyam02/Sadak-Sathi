## Tips for Great Screenshots

- Use high-resolution devices (at least 1080p)
- Capture in portrait mode
- Include realistic data (not "Test User" or lorem ipsum)
- Show real street names and locations
- Blur sensitive information if needed
- Use consistent time/battery level across screenshots
- Consider using device frames for marketing materials

## Tools

- **Android:** Built-in screenshot tool (Power + Volume Down)
- **iOS:** Built-in screenshot tool (Side Button + Volume Up)
- **Device Frames:** [mockuphone.com](https://mockuphone.com)
- **Editing:** Figma, Sketch, or Photoshop for annotations
